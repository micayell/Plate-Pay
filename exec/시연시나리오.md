# 📌 시연 시나리오

## 📱 모바일 앱

### 1. 로그인
1. SSAFY 계정으로 로그인
2. 로그인 성공 시 메인 화면(홈 화면) 진입

### 2. 계좌 관리
- **계좌 조회**
  1. 계좌 목록 조회 화면 진입
  2. 등록된 계좌 리스트 출력
  3. 월별 거래내역 통계 조회 가능  

- **계좌 추가**
  1. 계좌 추가하기 버튼 터치
  2. 계좌 별명, 계좌 번호 입력 후 등록 버튼 터치
  3. 1원 인증 숫자 4개 입력 후 인증 버튼 터치
  4. 인증 성공 시 계좌가 계좌 목록에 추가되어 표시됨

### 3. 차량 관리
- **차량 조회**
  1. 차량 목록 조회 화면 진입
  2. 등록된 차량 리스트 출력  

- **차량 추가**
  1. 차량 추가 버튼 클릭
  2. 차량 번호와 차량 별명 입력 후 다음 버튼 터치
  3. 이름과 주민등록번호 입력 후 인증 버튼 터치
  4. 카카오 인증 진행 후 인증 완료 버튼 터치
  5. 인증 성공 시 차량이 차량 목록에 추가되어 표시됨

### 4. 제휴 매장 탐색
1. 제휴된 주차장 리스트 조회 및 지도 마커 출력
2. 검색 버튼 터치 → 오버레이에서 카테고리 `카페` 선택
3. 키워드 `커피` 입력
4. 조건에 맞는 매장 및 주차장 리스트 출력, 지도에 마커 표시

---

## 🅿️ 바리게이트 (입차)
1. 차량을 직진시켜 카메라가 차량 번호 인식
2. 차량 번호가 등록되어 있으면 자동으로 차단기가 열림
3. 입차 완료

---

## 🏪 키오스크 (주문)
1. 제휴 매장 키오스크에서 상품 선택
2. 최종 선택된 상품들을 재확인
3. 결제 수단 선택에서 `플페이` 선택
4. 차량 번호 입력
5. 입력한 차량 번호와 일치하는 차량 리스트에서 내 차량 선택
6. 결제 방법 선택 → 결제 비밀번호 입력 또는 얼굴 인증 중 `얼굴 인증` 선택
7. 얼굴 인증 완료 시 주문 완료 및 결제 예정 내역 생성

---

1. 홈 화면에서 키오스크 주문 내역(결제 예정 내역) 확인 가능
